# ⚡ Power Distribution Board (PDB) | Hexapod v1

This folder contains the design files for the custom Power PCB designed to channel high-current rails to the 24 SC15 servos while providing regulated logic power to the ESP32.


---

## 🛠️ Technical Specifications

### 1. Power Rails
* **Input Voltage:** 7.4V - 12.6V (Compatible with 2S and 3S LiPo).
* **Servo Rail (Primary):** Direct high-current path from battery to daisy-chained SC15. 



### 2. Design Features
* **Heavy Copper Traces:** Designed with increased trace width/thickness to handle up to **15A peak current**.
